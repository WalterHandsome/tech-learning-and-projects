## 1. cProfile

### 1.1 基础使用

```python
import cProfile
import pstats

def slow_function():
    total = 0
    for i in range(1000000):
        total += i
    return total

# 性能分析
profiler = cProfile.Profile()
profiler.enable()
slow_function()
profiler.disable()

# 查看结果
stats = pstats.Stats(profiler)
stats.sort_stats('cumulative')
stats.print_stats(10)  # 显示前10个
```

### 1.2 命令行使用

```bash
python -m cProfile -o profile.stats script.py
python -m pstats profile.stats
```

## 2. line_profiler

### 2.1 行级分析

```python
# 安装: pip install line_profiler

@profile
def slow_function():
    total = 0
    for i in range(1000000):
        total += i
    return total

# 运行: kernprof -l -v script.py
```

## 3. memory_profiler

### 3.1 内存分析

```python
# 安装: pip install memory-profiler

@profile
def memory_intensive():
    data = [i for i in range(1000000)]
    return sum(data)

# 运行: python -m memory_profiler script.py
```

## 4. py-spy

### 4.1 实时分析

```bash
# 安装
pip install py-spy

# 实时分析
py-spy record -o profile.svg --pid 12345

# 查看实时统计
py-spy top --pid 12345
```

## 5. 总结

性能分析工具：
- **cProfile**：函数级性能分析
- **line_profiler**：行级性能分析
- **memory_profiler**：内存使用分析
- **py-spy**：实时性能分析

使用这些工具可以快速定位性能瓶颈。



## 6. Scalene — AI 驱动的性能分析器

> 🔄 更新于 2026-05-02

<!-- version-check: Scalene 1.5.x, py-spy, cProfile, checked 2026-05-02 -->

Scalene 是 2026 年最推荐的 Python 性能分析器，支持 CPU、GPU、内存的同时分析，并内置 AI 优化建议。来源：[Scalene GitHub](https://github.com/plasma-umass/scalene)

### 6.1 安装与使用

```bash
# 安装
pip install scalene

# 基础使用 — 同时分析 CPU + 内存
scalene script.py

# 生成 HTML 报告
scalene --html --outfile report.html script.py

# 只分析 CPU
scalene --cpu-only script.py

# 分析特定函数
scalene --profile-only "module.function_name" script.py
```

### 6.2 核心优势

```python
# Scalene 区分 Python 代码和 C 扩展的时间消耗
# 这是 cProfile 和 line_profiler 做不到的

# 示例输出：
#          CPU (Python)  CPU (C)  Memory
# Line 10:    45%          5%     +50MB
# Line 15:     2%         40%     +2MB
# Line 20:    10%          0%     -30MB

# Line 10 是纯 Python 热点 → 考虑用 NumPy 向量化
# Line 15 是 C 扩展热点 → 已经很快，优化空间小
```

### 6.3 AI 优化建议

```bash
# 启用 AI 优化建议（需要 OpenAI API Key）
scalene --ai script.py

# Scalene 会分析性能热点，自动生成优化建议
# 例如："Line 42 的列表推导可以替换为 NumPy 向量化操作，预计提速 10x"
```

### 6.4 2026 年性能分析工具选型

| 工具 | 分析维度 | 开销 | 适用场景 |
|------|---------|------|---------|
| **Scalene** | CPU + 内存 + GPU | 10-20% | 通用分析（推荐首选） |
| **cProfile** | 函数级 CPU | 低 | 快速定位慢函数 |
| **line_profiler** | 行级 CPU | 中 | 精确定位热点行 |
| **py-spy** | 采样 CPU | 极低 | 生产环境实时分析 |
| **memory_profiler** | 行级内存 | 高 | 内存泄漏排查 |
| **memray** | 内存分配追踪 | 中 | 详细内存分析 |

来源：[Scalene: Triangulating Python Performance Issues](https://arxiv.org/abs/2212.07597)
