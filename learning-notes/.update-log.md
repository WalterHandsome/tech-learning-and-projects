3. **iOS 26 BGContinuedProcessingTask 填补后台处理空白**：允许用户发起的长任务（视频导出、文件处理）在应用进入后台后继续执行，配合 Background GPU Access 支持 GPU 密集型任务。来源：[WWDC25 Session 227](https://developer.apple.com/videos/play/wwdc2025/227/)
4. **iOS 26 UIScene 生命周期迁移即将强制**：iOS 26 开始警告，iOS 27 将强制要求。推送通知处理需要迁移到 Scene 级别。来源：[Courier 迁移指南](https://courier.com/blog/ios-26-push-notification-changes-uiscene-requirment-ios-27)
5. **Liquid Glass 是 iOS 7 以来最大视觉重设计**：系统控件在 Xcode 26 编译后自动获得新外观，自定义 UI 通过 `.glassEffect()` 修饰符应用。设计原则：Glass 用于浮在内容之上的元素。来源：[Apple WWDC25 Session 284](https://developer.apple.com/videos/play/wwdc2025/284/)
6. **Elasticsearch 9.3.3 是当前稳定版**：GPU 向量索引（NVIDIA cuVS 集成，12x 索引加速）、Agent Builder GA、ES|QL Lookup Join 是核心新特性。Java 客户端引入 Rest5Client 替代旧 RestClient。来源：[Elastic 9.3 发布公告](https://www.elastic.co/blog/whats-new-elastic-9-3-0)
7. **Google March 2026 Core Update 改变 CWV 评估方式**：从页面级评估变为站点级聚合，受影响站点流量下降 20-35%。INP 仍是最常失败的指标（43% 站点未通过 200ms 阈值）。来源：[March 2026 Update Impact](https://logoswebdesigns.com/blog/core-web-vitals-2026-march-update/)
