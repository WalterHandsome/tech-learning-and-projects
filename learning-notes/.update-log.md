### 关键发现

1. **CSS Anchor Positioning 跨浏览器可用**（2026-01 起）：Chrome 125+、Firefox 132+、Safari 18.2+，覆盖约 91% 浏览器流量。可以替代 Popper.js/Floating UI 等 JavaScript 定位库。来源：[botmonster.com](https://botmonster.com/posts/css-anchor-positioning-tooltips-popovers/)
2. **CSS 原生嵌套、`:has()`、`@layer`、Container Queries 全部达到 ~95%+ 覆盖率**：2026 年这些特性已经是生产级可用，Sass/Less 预处理器的核心价值（嵌套）被原生 CSS 替代。来源：[Can I Use](https://caniuse.com)
3. **`@starting-style` 实现纯 CSS 入场动画**：配合 `allow-discrete` 关键字，可以对 `display: none` → `display: block` 的切换做过渡动画，替代 JavaScript 动画库。来源：[MDN @starting-style](https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/At-rules/@starting-style)
4. **Redis 8.6 Stream 幂等性是重大特性**：`XADD IDMPAUTO` 提供 at-most-once 投递保证，解决了 Redis Streams 长期以来的重复消费问题。来源：[Redis 8.6 What's New](https://redis.io/docs/latest/develop/whats-new/8-6/)
5. **Redis 8 统一发行版消除了模块碎片化**：JSON、Search、TimeSeries、Bloom 等全部内置，不再需要 Redis Stack。来源：[Redis 8.0 What's New](https://redis.io/docs/latest/develop/whats-new/8-0/)
6. **Spring Cloud OpenFeign 在 2025.1 中被移除**：HTTP Interface Client（`@HttpExchange`）成为替代方案，这是 2026 年 Java 面试高频考点。
7. **Virtual Threads pinning 问题是面试陷阱**：`synchronized` 会导致虚拟线程被固定到 OS 线程，必须改用 `ReentrantLock`。来源：[backendbytes.com](https://backendbytes.com/articles/java-virtual-threads-concurrency-revolution/)
