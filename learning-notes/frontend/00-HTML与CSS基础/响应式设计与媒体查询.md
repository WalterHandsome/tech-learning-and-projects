# 响应式设计与媒体查询

## 1. 视口设置

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

## 2. 媒体查询

### 2.1 基本语法

```css
/* 最大宽度（移动优先推荐用 min-width） */
@media (max-width: 768px) {
  .sidebar { display: none; }
}

/* 最小宽度（移动优先策略） */
@media (min-width: 768px) {
  .container { max-width: 720px; }
}
@media (min-width: 1024px) {
  .container { max-width: 960px; }
}
@media (min-width: 1280px) {
  .container { max-width: 1200px; }
}

/* 组合条件 */
@media (min-width: 768px) and (max-width: 1023px) {
  /* 平板样式 */
}

/* 暗色模式 */
@media (prefers-color-scheme: dark) {
  body { background: #1a1a2e; color: #eee; }
}

/* 减少动画 */
@media (prefers-reduced-motion: reduce) {
  * { animation: none !important; transition: none !important; }
}
```

### 2.2 常用断点

```css
/* 移动优先断点 */
/* 默认样式：手机（< 640px） */
@media (min-width: 640px)  { /* 大手机/小平板 */ }
@media (min-width: 768px)  { /* 平板 */ }
@media (min-width: 1024px) { /* 笔记本 */ }
@media (min-width: 1280px) { /* 桌面 */ }
@media (min-width: 1536px) { /* 大屏 */ }
```

## 3. 响应式单位

```css
.responsive {
  /* 相对单位 */
  font-size: 1rem;      /* 相对根元素字体大小 */
  padding: 1em;         /* 相对当前元素字体大小 */
  width: 50vw;          /* 视口宽度的50% */
  height: 100vh;        /* 视口高度的100% */
  height: 100dvh;       /* 动态视口高度（移动端推荐） */

  /* clamp() 响应式字体 */
  font-size: clamp(14px, 2.5vw, 20px);

  /* 容器查询单位 */
  font-size: 5cqi;      /* 容器内联尺寸的5% */
}
```

## 4. 响应式图片

```html
<!-- srcset + sizes -->
<img
  src="image-800.jpg"
  srcset="image-400.jpg 400w, image-800.jpg 800w, image-1200.jpg 1200w"
  sizes="(max-width: 600px) 100vw, (max-width: 1024px) 50vw, 33vw"
  alt="响应式图片"
>

<!-- picture 元素 -->
<picture>
  <source media="(min-width: 1024px)" srcset="large.webp" type="image/webp">
  <source media="(min-width: 768px)" srcset="medium.webp" type="image/webp">
  <img src="small.jpg" alt="响应式图片">
</picture>
```

## 5. 容器查询（Container Queries）

```css
/* 定义容器 */
.card-container {
  container-type: inline-size;
  container-name: card;
}

/* 基于容器宽度的样式 */
@container card (min-width: 400px) {
  .card {
    display: flex;
    flex-direction: row;
  }
}

@container card (max-width: 399px) {
  .card {
    display: flex;
    flex-direction: column;
  }
}
```
