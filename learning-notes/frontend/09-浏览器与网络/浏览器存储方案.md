# 浏览器存储方案

## 1. 存储方案对比

| 方案 | 容量 | 生命周期 | 作用域 | 同步/异步 |
|------|------|---------|--------|----------|
| Cookie | ~4KB | 可设过期时间 | 同源+路径 | 同步 |
| localStorage | ~5-10MB | 永久 | 同源 | 同步 |
| sessionStorage | ~5-10MB | 标签页关闭 | 同源+标签页 | 同步 |
| IndexedDB | 无限制 | 永久 | 同源 | 异步 |
| Cache API | 无限制 | 手动管理 | 同源 | 异步 |

## 2. Cookie

```javascript
// 设置 Cookie
document.cookie = 'name=张三; max-age=3600; path=/; Secure; SameSite=Lax';

// 读取 Cookie
function getCookie(name) {
  const match = document.cookie.match(new RegExp(`(^| )${name}=([^;]*)`));
  return match ? decodeURIComponent(match[2]) : null;
}

// 删除 Cookie
document.cookie = 'name=; max-age=0; path=/';

// Cookie 属性
// HttpOnly  — JS 不可访问（防 XSS）
// Secure    — 仅 HTTPS 传输
// SameSite  — Strict/Lax/None（防 CSRF）
// Domain    — 作用域名
// Path      — 作用路径
// Max-Age   — 过期秒数
// Expires   — 过期日期
```

## 3. localStorage / sessionStorage

```javascript
// 封装（带过期时间）
const storage = {
  set(key, value, ttl) {
    const item = { value, expire: ttl ? Date.now() + ttl : null };
    localStorage.setItem(key, JSON.stringify(item));
  },
  get(key) {
    const raw = localStorage.getItem(key);
    if (!raw) return null;
    const item = JSON.parse(raw);
    if (item.expire && Date.now() > item.expire) {
      localStorage.removeItem(key);
      return null;
    }
    return item.value;
  },
  remove(key) { localStorage.removeItem(key); },
  clear() { localStorage.clear(); },
};

// 监听存储变化（跨标签页通信）
window.addEventListener('storage', (e) => {
  console.log(`${e.key}: ${e.oldValue} → ${e.newValue}`);
});
```

## 4. IndexedDB

```javascript
// 使用 idb 库简化操作
import { openDB } from 'idb';

const db = await openDB('myDB', 1, {
  upgrade(db) {
    const store = db.createObjectStore('users', { keyPath: 'id', autoIncrement: true });
    store.createIndex('email', 'email', { unique: true });
  },
});

// CRUD
await db.add('users', { name: '张三', email: 'test@example.com' });
const user = await db.get('users', 1);
const all = await db.getAll('users');
await db.put('users', { id: 1, name: '李四', email: 'test@example.com' });
await db.delete('users', 1);

// 按索引查询
const user = await db.getFromIndex('users', 'email', 'test@example.com');
```

## 5. 选型建议

- 认证信息（Token）→ HttpOnly Cookie（最安全）或 localStorage
- 用户偏好（主题、语言）→ localStorage
- 表单临时数据 → sessionStorage
- 大量结构化数据 → IndexedDB
- 离线资源缓存 → Cache API + Service Worker
