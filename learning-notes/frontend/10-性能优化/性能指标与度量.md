# 性能指标与度量

> Author: Walter Wang

## 1. Core Web Vitals

```
LCP（Largest Contentful Paint）— 最大内容绘制
  好: < 2.5s | 需改进: 2.5-4s | 差: > 4s
  衡量加载性能，最大可见元素的渲染时间

INP（Interaction to Next Paint）— 交互到下一次绘制
  好: < 200ms | 需改进: 200-500ms | 差: > 500ms
  衡量交互响应性（替代 FID）

CLS（Cumulative Layout Shift）— 累积布局偏移
  好: < 0.1 | 需改进: 0.1-0.25 | 差: > 0.25
  衡量视觉稳定性
```

## 2. 其他指标

```
FCP（First Contentful Paint）— 首次内容绘制
TTFB（Time to First Byte）— 首字节时间
TTI（Time to Interactive）— 可交互时间
TBT（Total Blocking Time）— 总阻塞时间
```

## 3. 度量工具

```javascript
// Performance API
const [entry] = performance.getEntriesByType('navigation');
console.log('TTFB:', entry.responseStart - entry.requestStart);
console.log('DOM解析:', entry.domContentLoadedEventEnd - entry.responseEnd);
console.log('页面加载:', entry.loadEventEnd - entry.navigationStart);

// Web Vitals 库
import { onLCP, onINP, onCLS } from 'web-vitals';

onLCP(({ value }) => console.log('LCP:', value));
onINP(({ value }) => console.log('INP:', value));
onCLS(({ value }) => console.log('CLS:', value));

// 上报到监控平台
function sendToAnalytics(metric) {
  navigator.sendBeacon('/api/metrics', JSON.stringify(metric));
}
onLCP(sendToAnalytics);
onINP(sendToAnalytics);
onCLS(sendToAnalytics);
```
