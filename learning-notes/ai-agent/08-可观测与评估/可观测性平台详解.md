## 2. LangSmith

LangChain 官方可观测性平台，与 LangGraph 深度集成。

```python
# 零配置集成（设置环境变量即可）
import os
os.environ["LANGSMITH_API_KEY"] = "lsv2_xxx"
os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_PROJECT"] = "my-agent"

# LangGraph Agent 自动追踪
from langgraph.prebuilt import create_react_agent
from langchain_openai import ChatOpenAI

agent = create_react_agent(ChatOpenAI(model="gpt-4o"), tools=[...])
result = agent.invoke({"messages": [("user", "你好")]})
# → LangSmith 自动记录完整 trace

# 手动追踪（非 LangChain 代码）
from langsmith import traceable

@traceable(name="custom_agent_step")
def my_agent_step(query: str) -> str:
    response = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": query}],
    )
    return response.choices[0].message.content

# 在线评估
from langsmith import Client

ls_client = Client()
ls_client.create_feedback(
    run_id="run-xxx",
    key="correctness",
    score=0.9,
    comment="回答准确",
)
```

```
LangSmith 核心功能：
├─ Trace 追踪：完整调用链可视化
├─ Prompt Hub：提示词版本管理
├─ 在线评估：对生产 trace 打分
├─ 数据集管理：构建评估数据集
├─ A/B 测试：对比不同配置效果
└─ 近零开销：<1ms 额外延迟
```

## 3. LangFuse

开源可观测性平台，支持多框架，可自托管。

```python
# pip install langfuse
from langfuse import Langfuse
from langfuse.decorators import observe, langfuse_context

langfuse = Langfuse(
    public_key="pk-xxx",
    secret_key="sk-xxx",
    host="https://cloud.langfuse.com",  # 或自托管地址
)

# 装饰器方式追踪
@observe()
def agent_pipeline(query: str) -> str:
    # 自动追踪函数调用
    research = research_step(query)
    answer = generate_answer(research)
    return answer

@observe()
def research_step(query: str) -> str:
    # 记录 LLM 调用
    langfuse_context.update_current_observation(
        input=query,
        metadata={"step": "research"},
    )
    return "调研结果..."

# OpenAI 集成（自动追踪所有调用）
from langfuse.openai import openai
client = openai.OpenAI()
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "你好"}],
)
# → 自动记录到 LangFuse

# 成本追踪
# LangFuse 自动计算每次调用的 Token 成本
# Dashboard 展示：日/周/月成本趋势
```

```bash
# 自托管部署（Docker）
docker compose up -d
# 访问 http://localhost:3000
```

## 4. Arize Phoenix

开源可观测性平台，专注嵌入分析和漂移检测。

```python
# pip install arize-phoenix
import phoenix as px

# 启动 Phoenix 服务
session = px.launch_app()
# → http://localhost:6006

# OpenAI 集成
from phoenix.otel import register
from openinference.instrumentation.openai import OpenAIInstrumentor

tracer_provider = register(project_name="my-agent")
OpenAIInstrumentor().instrument(tracer_provider=tracer_provider)

# 所有 OpenAI 调用自动追踪
from openai import OpenAI
client = OpenAI()
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "你好"}],
)

# LangChain 集成
from openinference.instrumentation.langchain import LangChainInstrumentor
LangChainInstrumentor().instrument(tracer_provider=tracer_provider)
```

```
Arize Phoenix 特色功能：
├─ 嵌入聚类：可视化 embedding 分布
├─ 漂移检测：发现数据分布变化
├─ LLM 评估：内置评估器（幻觉、相关性等）
├─ RAG 分析：检索质量评估
└─ 完全开源：Apache 2.0
```

## 5. AgentOps

专为 Agent 设计的可观测性平台，CrewAI 原生集成。

```python
import agentops

# 初始化（自动追踪）
agentops.init(api_key="ao-xxx")

# CrewAI 自动集成
from crewai import Agent, Task, Crew

researcher = Agent(role="研究员", goal="调研", llm="gpt-4o")
task = Task(description="调研 AI Agent", agent=researcher)
crew = Crew(agents=[researcher], tasks=[task])
result = crew.kickoff()
# → AgentOps 自动记录完整 Agent 会话

# 手动记录事件
agentops.record(agentops.ActionEvent(action_type="search", params={"query": "AI"}))

# 结束会话
agentops.end_session("Success")
```

```
AgentOps 特色：
├─ Session Replay：回放 Agent 完整执行过程
├─ Agent Analytics：Agent 级别的统计分析
├─ CrewAI 原生：零配置集成
├─ 成本追踪：按 Agent/Session 统计
└─ 错误追踪：自动捕获异常
```

## 6. Helicone

代理模式可观测性，零代码集成。

```python
from openai import OpenAI

# 只需修改 base_url，零代码集成
client = OpenAI(
    base_url="https://oai.helicone.ai/v1",
    default_headers={
        "Helicone-Auth": "Bearer hl-xxx",
        "Helicone-User-Id": "alice",
        "Helicone-Session-Id": "session-001",
    },
)

# 所有请求自动经过 Helicone 代理记录
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "你好"}],
)

# 启用缓存（相同请求直接返回缓存）
client_cached = OpenAI(
    base_url="https://oai.helicone.ai/v1",
    default_headers={
        "Helicone-Auth": "Bearer hl-xxx",
        "Helicone-Cache-Enabled": "true",
    },
)
```

## 7. Braintrust

评估优先的可观测性平台。

```python
from braintrust import Eval, init_logger

# 评估驱动开发
await Eval(
    "my-agent",
    data=lambda: [
        {"input": "什么是 RAG？", "expected": "检索增强生成"},
        {"input": "LangGraph 是什么？", "expected": "工作流编排框架"},
    ],
    task=lambda input: my_agent(input),
    scores=[
        lambda output, expected: {
            "name": "correctness",
            "score": 1.0 if expected.lower() in output.lower() else 0.0,
        },
    ],
)

# 生产日志
logger = init_logger(project="my-agent")
logger.log(input="用户问题", output="Agent回答", scores={"quality": 0.9})
```

## 10. 选型建议
