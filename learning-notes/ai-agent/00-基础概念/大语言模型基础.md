# 大语言模型基础

## 1. Transformer 架构

Transformer 是现代大语言模型的基础架构，核心机制是自注意力（Self-Attention）。

```
输入文本 → Tokenization → Embedding → [Encoder/Decoder Blocks] → 输出

Encoder Block:
  Multi-Head Self-Attention → Add & Norm → Feed Forward → Add & Norm

Decoder Block:
  Masked Self-Attention → Cross-Attention → Feed Forward → 输出概率分布
```

```python
# 注意力机制核心公式的直觉理解
# Attention(Q, K, V) = softmax(QK^T / sqrt(d_k)) * V

import torch
import torch.nn.functional as F

def scaled_dot_product_attention(Q, K, V):
    """缩放点积注意力"""
    d_k = Q.size(-1)
    scores = torch.matmul(Q, K.transpose(-2, -1)) / (d_k ** 0.5)
    weights = F.softmax(scores, dim=-1)
    return torch.matmul(weights, V)
```

## 3. Token 与上下文窗口

```python
import tiktoken

# GPT 系列使用 tiktoken
enc = tiktoken.encoding_for_model("gpt-4o")
text = "AI Agent 是一个能够自主决策的智能系统"
tokens = enc.encode(text)
print(f"文本: {text}")
print(f"Token 数: {len(tokens)}")  # 中文通常 1 字 ≈ 1-2 tokens
print(f"Token IDs: {tokens}")
print(f"解码: {enc.decode(tokens)}")

# 上下文窗口 = 输入 tokens + 输出 tokens
# GPT-4o: 128K 输入 + 16K 输出
# Claude 3.5: 200K 输入 + 8K 输出
```

## 4. 采样参数

```python
from openai import OpenAI
client = OpenAI()

# Temperature: 控制随机性（0=确定性，2=高随机性）
# Top-P: 核采样，从概率累积前 P 的 token 中采样
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "写一首关于AI的诗"}],
    temperature=0.7,   # 创意任务用 0.7-1.0
    top_p=0.9,         # 通常与 temperature 二选一调整
    max_tokens=500,
    frequency_penalty=0.5,  # 减少重复（0-2）
    presence_penalty=0.3,   # 鼓励新话题（0-2）
)

# 不同场景推荐参数
# 代码生成: temperature=0, top_p=1
# 创意写作: temperature=0.9, top_p=0.95
# 数据提取: temperature=0, top_p=1
# 对话聊天: temperature=0.7, top_p=0.9
```
