# 开源模型部署

## 1. 推理引擎对比

| 引擎 | 特点 | 适用场景 |
|------|------|---------|
| vLLM | 高吞吐、PagedAttention | 生产级 API 服务 |
| Ollama | 一键部署、极简操作 | 本地开发、个人使用 |
| llama.cpp | CPU/GPU 混合推理、GGUF | 边缘设备、低资源 |
| TGI | HuggingFace 官方、容器化 | HF 生态集成 |

## 2. Ollama 本地部署

```bash
# 安装 Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 下载并运行模型
ollama pull llama3.1:8b
ollama pull qwen2.5:14b
ollama run llama3.1:8b "什么是 AI Agent？"

# 查看已下载模型
ollama list
```

```python
# Python 调用 Ollama（兼容 OpenAI API）
from openai import OpenAI

client = OpenAI(base_url="http://localhost:11434/v1", api_key="ollama")

response = client.chat.completions.create(
    model="qwen2.5:14b",
    messages=[
        {"role": "system", "content": "你是一个有帮助的助手"},
        {"role": "user", "content": "解释 RAG 的核心流程"},
    ],
    temperature=0.7,
)
print(response.choices[0].message.content)
```

## 3. vLLM 生产部署

```bash
# 安装 vLLM
pip install vllm

# 启动 API 服务
python -m vllm.entrypoints.openai.api_server \
    --model Qwen/Qwen2.5-14B-Instruct \
    --tensor-parallel-size 2 \
    --max-model-len 32768 \
    --gpu-memory-utilization 0.9 \
    --port 8000
```

```python
# vLLM 兼容 OpenAI API
from openai import OpenAI

client = OpenAI(base_url="http://localhost:8000/v1", api_key="vllm")

response = client.chat.completions.create(
    model="Qwen/Qwen2.5-14B-Instruct",
    messages=[{"role": "user", "content": "你好"}],
    temperature=0.7,
    max_tokens=500,
)
```

## 4. llama.cpp（GGUF 量化推理）

```bash
# 安装 llama-cpp-python
pip install llama-cpp-python

# 下载 GGUF 量化模型
# huggingface-cli download TheBloke/Llama-3.1-8B-Instruct-GGUF \
#   llama-3.1-8b-instruct.Q4_K_M.gguf
```

```python
from llama_cpp import Llama

llm = Llama(
    model_path="./models/llama-3.1-8b-instruct.Q4_K_M.gguf",
    n_ctx=8192,       # 上下文长度
    n_gpu_layers=35,  # GPU 加速层数（-1 = 全部）
    n_threads=8,      # CPU 线程数
)

output = llm.create_chat_completion(
    messages=[{"role": "user", "content": "什么是 MCP 协议？"}],
    temperature=0.7,
    max_tokens=500,
)
print(output["choices"][0]["message"]["content"])
```

## 5. 量化方案对比

| 量化方式 | 精度损失 | 速度 | 显存占用 | 适用场景 |
|---------|---------|------|---------|---------|
| FP16 | 无 | 基准 | 100% | 精度优先 |
| GGUF Q8 | 极小 | 快 | ~50% | 质量与速度平衡 |
| GGUF Q4_K_M | 小 | 很快 | ~30% | 推荐默认选择 |
| AWQ | 小 | 快 | ~30% | vLLM 生产部署 |
| GPTQ | 小 | 快 | ~30% | TGI 部署 |

## 7. GPU 资源规划

```python
def estimate_gpu_memory(param_billions: float, precision: str = "fp16") -> float:
    """估算模型显存需求（GB）"""
    bytes_per_param = {"fp32": 4, "fp16": 2, "int8": 1, "int4": 0.5}
    model_size = param_billions * bytes_per_param[precision]
    # 推理额外开销约 20%
    return model_size * 1.2

# 示例
print(f"Qwen2.5-14B FP16: {estimate_gpu_memory(14, 'fp16'):.0f} GB")   # ~34 GB
print(f"Qwen2.5-14B INT4: {estimate_gpu_memory(14, 'int4'):.0f} GB")   # ~8 GB
print(f"Llama-70B INT4:   {estimate_gpu_memory(70, 'int4'):.0f} GB")   # ~42 GB

# GPU 选型参考
# RTX 4090 (24GB): 14B FP16 / 70B INT4（勉强）
# A100 (80GB):     70B FP16 / 405B INT4
# H100 (80GB):     70B FP16（更快）
```
