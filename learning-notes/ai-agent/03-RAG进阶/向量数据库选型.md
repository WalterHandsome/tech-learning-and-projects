# 向量数据库选型

## 1. 主流向量数据库对比

| 特性 | Pinecone | Weaviate | Qdrant | Milvus | Chroma | pgvector |
|------|----------|----------|--------|--------|--------|----------|
| 类型 | 全托管 | 自托管/云 | 自托管/云 | 自托管/云 | 嵌入式 | PG 扩展 |
| 语言 | - | Go | Rust | Go/C++ | Python | C |
| 混合搜索 | ✅ | ✅ | ✅ | ✅ | ❌ | 有限 |
| 过滤 | 元数据 | GraphQL | 丰富 | 丰富 | 基础 | SQL |
| 扩展性 | 自动 | 水平 | 水平 | 水平 | 单机 | 单机 |
| 适用场景 | 生产SaaS | 语义搜索 | 高性能 | 大规模 | 原型开发 | 已有PG |
| 学习曲线 | 低 | 中 | 中 | 高 | 极低 | 低 |

## 2. Chroma（轻量开发）

```python
import chromadb
from chromadb.utils import embedding_functions

# 创建客户端
client = chromadb.PersistentClient(path="./chroma_db")

# 使用 OpenAI Embedding
ef = embedding_functions.OpenAIEmbeddingFunction(
    api_key="sk-xxx",
    model_name="text-embedding-3-small",
)

# 创建集合
collection = client.get_or_create_collection(
    name="documents",
    embedding_function=ef,
    metadata={"hnsw:space": "cosine"},  # 距离度量
)

# 添加文档
collection.add(
    documents=["AI Agent 是自主决策的智能系统", "RAG 结合检索与生成"],
    metadatas=[{"source": "doc1"}, {"source": "doc2"}],
    ids=["id1", "id2"],
)

# 查询
results = collection.query(query_texts=["什么是Agent"], n_results=3)
print(results["documents"])
```

## 3. Qdrant（高性能生产）

```python
from qdrant_client import QdrantClient
from qdrant_client.models import Distance, VectorParams, PointStruct, Filter, FieldCondition

client = QdrantClient(host="localhost", port=6333)

# 创建集合
client.create_collection(
    collection_name="knowledge_base",
    vectors_config=VectorParams(size=1536, distance=Distance.COSINE),
)

# 插入向量
client.upsert(
    collection_name="knowledge_base",
    points=[
        PointStruct(id=1, vector=embedding_1, payload={"text": "...", "source": "doc1"}),
        PointStruct(id=2, vector=embedding_2, payload={"text": "...", "source": "doc2"}),
    ],
)

# 带过滤的搜索
results = client.query_points(
    collection_name="knowledge_base",
    query=query_vector,
    query_filter=Filter(must=[FieldCondition(key="source", match={"value": "doc1"})]),
    limit=5,
)
```

## 4. pgvector（PostgreSQL 扩展）

```python
# 适合已有 PostgreSQL 基础设施的团队
import psycopg2

conn = psycopg2.connect("postgresql://localhost/mydb")
cur = conn.cursor()

# 启用扩展
cur.execute("CREATE EXTENSION IF NOT EXISTS vector")

# 创建表
cur.execute("""
    CREATE TABLE IF NOT EXISTS documents (
        id SERIAL PRIMARY KEY,
        content TEXT,
        metadata JSONB,
        embedding vector(1536)
    )
""")

# 插入
cur.execute(
    "INSERT INTO documents (content, metadata, embedding) VALUES (%s, %s, %s)",

# 相似度搜索（余弦距离）
cur.execute("""
    SELECT content, 1 - (embedding <=> %s::vector) AS similarity
    FROM documents
    ORDER BY embedding <=> %s::vector
    LIMIT 5
""", (str(query_vector), str(query_vector)))
```

## 5. 混合搜索（Vector + Keyword + Semantic）

```python
# Qdrant 混合搜索示例
from qdrant_client.models import SparseVector

# 同时使用稠密向量（语义）和稀疏向量（关键词）
client.create_collection(
    collection_name="hybrid_search",

# 混合查询
results = client.query_points(

```

## 6. 选型指南

```
├── 快速原型 / 小数据量？
│   └── Chroma（嵌入式，零配置）
├── 已有 PostgreSQL？
│   └── pgvector（无需新增组件）
├── 需要高性能 + 丰富过滤？
│   └── Qdrant（Rust 实现，性能优秀）
├── 大规模（亿级向量）？
│   └── Milvus（分布式，GPU 加速）
├── 全托管 + 零运维？
│   └── Pinecone（SaaS，自动扩展）
└── 语义搜索 + GraphQL？
    └── Weaviate（模块化，多模态）
```
