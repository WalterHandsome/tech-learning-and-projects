## 1. 百炼平台概述

阿里云百炼是一站式 AI Agent 构建平台，提供模型调用、知识库管理、插件市场、工作流编排等能力。支持通义千问全系列模型，也可接入第三方模型。

```
┌──────────────────── 百炼平台 ────────────────────┐
│                                                    │
│  ┌──────────┐  ┌──────────┐  ┌────────────────┐  │
│  │ 模型服务  │  │ 知识库    │  │ 插件/工具      │  │
│  │ Qwen系列  │  │ 文档解析  │  │ 内置+自定义    │  │
│  │ 第三方模型 │  │ 向量检索  │  │ API 插件      │  │
│  └──────────┘  └──────────┘  └────────────────┘  │
│  ┌──────────────────────────────────────────────┐ │
│  │           工作流编排（可视化 + API）            │ │
│  └──────────────────────────────────────────────┘ │
│  ┌──────────────────────────────────────────────┐ │
│  │     DashScope API / OpenAI 兼容接口           │ │
│  └──────────────────────────────────────────────┘ │
└───────────────────────────────────────────────────┘
```

## 3. DashScope API 调用

```python
from openai import OpenAI

# 百炼兼容 OpenAI 接口
client = OpenAI(
    api_key="sk-xxx",
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
)

response = client.chat.completions.create(
    model="qwen-max",
    messages=[
        {"role": "system", "content": "你是一个有帮助的助手。"},
        {"role": "user", "content": "解释什么是 AI Agent"},
    ],
    temperature=0.7,
)
print(response.choices[0].message.content)
```

## 4. Function Calling

```python
import json
from openai import OpenAI

client = OpenAI(
    api_key="sk-xxx",
    base_url="https://dashscope.aliyuncs.com/compatible-mode/v1",
)

tools = [
    {
        "type": "function",
        "function": {
            "name": "get_stock_price",
            "description": "查询股票实时价格",
            "parameters": {
                "type": "object",
                "properties": {
                    "symbol": {"type": "string", "description": "股票代码，如 600519"},
                    "market": {"type": "string", "enum": ["sh", "sz"], "description": "交易所"},
                },
                "required": ["symbol"],
            },
        },
    }
]

response = client.chat.completions.create(
    model="qwen-max",
    messages=[{"role": "user", "content": "查一下茅台的股价"}],
    tools=tools,
    tool_choice="auto",
)

# 处理工具调用
message = response.choices[0].message
if message.tool_calls:
    for call in message.tool_calls:
        args = json.loads(call.function.arguments)
        print(f"调用: {call.function.name}, 参数: {args}")
        # → 调用: get_stock_price, 参数: {"symbol": "600519", "market": "sh"}
```

## 5. 百炼 Agent 构建

### 知识库 RAG

```python
# 通过 DashScope API 使用百炼知识库
response = client.chat.completions.create(
    model="qwen-max",
    messages=[{"role": "user", "content": "公司的退货政策是什么？"}],
    extra_body={
        "knowledge_base": {
            "knowledge_base_id": "kb-xxx",
            "top_k": 5,
            "score_threshold": 0.5,
        }
    },
)
```

### 工作流 Agent

```python
# 百炼工作流 API 调用
import dashscope

response = dashscope.Application.call(
    app_id="app-xxx",  # 百炼控制台创建的应用 ID
    prompt="帮我分析上个月的销售数据",
    session_id="session-001",  # 多轮对话
)
print(response.output.text)
```

## 7. 与 LangChain / LlamaIndex 集成

```python
# LangChain 集成
from langchain_community.llms import Tongyi

llm = Tongyi(model="qwen-max", dashscope_api_key="sk-xxx")
result = llm.invoke("什么是 RAG？")

# LlamaIndex 集成
from llama_index.llms.dashscope import DashScope

llm = DashScope(model_name="qwen-max", api_key="sk-xxx")
response = llm.complete("解释向量数据库的原理")
```

## 8. 私有化部署

```bash
# 使用 vLLM 部署 Qwen2.5-72B
pip install vllm
python -m vllm.entrypoints.openai.api_server \
    --model Qwen/Qwen2.5-72B-Instruct \
    --tensor-parallel-size 4 \
    --max-model-len 32768 \
    --port 8000

# 使用 Ollama 部署（轻量）
ollama pull qwen2.5:72b
ollama serve
```
