### 2. SpaceX 以 600 亿美元收购选择权锁定 Cursor：AI Coding Agent 估值天花板再次刷新

- **来源**：[36氪](https://36kr.com/p/3781532841581576) / [Bloomberg via Yahoo Finance](https://finance.yahoo.com/sectors/technology/articles/spacex-deal-acquire-cursor-60-003344236.html) / [Business Insider](https://www.businessinsider.com/ai-coding-startup-cursor-elon-musk-60-billion-spacex-deal-2026-4)
- **标签**：`SpaceX` `Cursor` `收购` `Coding Agent`
- **评分**：时效 5 / 一手 5 / 相关 5 / 实用 3 = 总分 18
- **摘要**：SpaceX（已与 xAI 合并）于 4 月 21 日宣布与 Cursor 母公司 Anysphere 达成协议：SpaceX 获得以 600 亿美元收购 Cursor 的选择权，若不行使则支付 100 亿美元终止费。Cursor 由四名 MIT 辍学的"00 后"于四年前创立，付费用户超 100 万，超半数《财富》500 强企业使用。此前 Cursor 正在进行 20 亿美元融资，估值从 2024 年初的 25 亿飙升至 2025 年底的 293 亿。Musk 此前承认 xAI 在编码工具上落后于竞争对手。
- **影响**：600 亿美元的收购选择权使 Cursor 成为 AI 历史上估值最高的 Coding Agent 公司。这笔交易的战略逻辑是 SpaceX/xAI 需要补齐 AI 编码能力短板——xAI 的 Grok 在编码基准上落后于 Claude 和 GPT-5.5。对中国开发者的启示：Coding Agent 赛道的估值天花板远超预期，国内类似产品（如通义灵码、豆包 MarsCode）的战略价值可能被低估。
- **行动建议**：关注 Cursor 在 SpaceX/xAI 体系下的产品演进，以及对 Claude Code、Codex 竞争格局的影响。

### 3. GitHub Copilot 暂停新注册：Agent 工作负载击穿原有定价模型

- **来源**：[InfoQ CN](https://www.infoq.cn/article/U3BlkMiBlgWgLiKFth4N) / [The Next Web](https://thenextweb.com/news/github-copilot-signup-pause-agentic-ai-usage-limits) / [blockchain.news](https://blockchain.news/news/github-pauses-copilot-signups-ai-agents-overwhelm-infrastructure)
- **标签**：`GitHub` `Copilot` `定价危机` `Agent`
- **评分**：时效 5 / 一手 5 / 相关 5 / 实用 4 = 总分 19
- **摘要**：GitHub 于 4 月 20 日宣布暂停 Copilot Pro、Pro+ 和 Student 计划的新用户注册，同时收紧现有用户的使用限额。原因是 Agent 工作负载（长时间运行的 Agent 会话、并行化工作流、高级模型需求）远超原有固定月费定价模型的承载能力。Claude Opus 4.5/4.6 从 Pro+ 计划中移除，仅保留在更高价位。GitHub 表示这是为了"更有效地服务现有客户"。5 月 20 日前不满意的用户可申请退款。
- **影响**：这是继 Anthropic 测试将 Claude Code 从 Pro 计划移除（04-23 简报）之后，又一个 Coding Agent 定价模型被击穿的案例。核心矛盾是：Agent 的计算成本远高于传统代码补全，固定月费模式不可持续。这预示着整个 Coding Agent 行业将从"无限量订阅"转向"按量计费"或"分层限额"模式。对国内开发者的影响：依赖 Copilot 的团队需要评估替代方案。
- **行动建议**：(1) 如果你是 Copilot 用户，检查新的使用限额是否影响工作流；(2) 评估 Claude Code、Codex、通义灵码等替代方案；(3) 关注 Copilot 后续定价调整。
