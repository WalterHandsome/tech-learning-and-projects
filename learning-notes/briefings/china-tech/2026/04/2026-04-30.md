### DeepSeek V4 实测：10 位从业者的真心话

- **来源**：[别急着 All-in DeepSeek V4，先看看这 10 位从业者的真心话](https://36kr.com/p/3788151000751364?f=rss)
- **标签**：`DeepSeek` `V4` `模型评测` `工程优化`
- **评分**：时效 5 / 一手 4 / 相关 5 / 实用 4 = 总分 18
- **摘要**：36 氪采访了 10 位 AI 从业者对 DeepSeek V4 的真实评价。V4 的核心突破在于"克制美学"——不靠堆算力，而是通过注意力机制、MoE 架构、后训练和推理系统工程的组合优化，将百万 Token 长上下文的算力需求压到 V3.2 的 27%，KV 缓存压缩到 10%。V4-Pro 在 SWE-bench 上达到 80.6%，接近 Claude Opus 4.6，但输出成本仅为其 1/7。
- **影响**：V4 的工程优化路线对国内 AI 开发者有直接参考价值——不一定要追求最大参数，精细化的架构优化同样能达到前沿水平。对于 Agent 场景，百万 Token 上下文 + 低成本推理是刚需。
- **行动建议**：关注 V4 的 API 正式开放时间；如果你的 Agent 项目对长上下文有需求，V4 值得作为候选底座模型评估。

### 蚂蚁百灵 Ling-2.6-flash 开源：Agent 场景的效率之王

- **来源**：[Ling-2.6-flash on Hugging Face](https://huggingface.co/inclusionAI/Ling-2.6-flash)
- **标签**：`蚂蚁集团` `百灵` `开源模型` `MoE`
- **评分**：时效 5 / 一手 5 / 相关 4 / 实用 4 = 总分 18
- **摘要**：蚂蚁集团正式开源百灵大模型 Ling-2.6-flash，总参数 104B，激活参数仅 7.4B。采用混合线性架构，在 H20 GPU 上推理速度可达 340 tokens/s。在 Artificial Analysis 完整评估中仅消耗 15M tokens，输出成本约为同类模型的 1/4。同时开源了 Ling-2.6-1T 万亿参数旗舰版。
- **影响**：Ling-2.6-flash 明确针对 Agent 场景设计——Agent 工作流涉及大量输入和多步执行，token 消耗是部署的主要障碍。这个模型的效率优势在 Agent 场景中尤为突出。
- **行动建议**：如果你在做 Agent 项目且对推理成本敏感，值得在 Hugging Face 上拉取 Ling-2.6-flash 进行评估，尤其是 FP8 和 INT4 量化版本。

- **标签**：`腾讯` `ima` `知识Agent` `个人记忆`
- **评分**：时效 5 / 一手 4 / 相关 5 / 实用 3 = 总分 17
- **摘要**：腾讯智能工作台 ima 正式上线个人知识 Agent「copilot」，内置记忆系统包含四个模块：copilot 设置、用户画像、长期记忆、经验与技巧。能够记住用户背景、习惯和待办事项，实现跨场景持续调用，减少重复输入。

### CVE-2026-31431：影响 17 年 Linux 内核的本地提权漏洞

- **来源**：[V2EX 讨论](https://www.v2ex.com/t/1209523#reply19) / [The Register](https://www.theregister.com/2026/04/30/linux_cryptographic_code_flaw/)
- **标签**：`Linux` `CVE` `本地提权` `安全`
- **评分**：时效 5 / 一手 4 / 相关 4 / 实用 4 = 总分 17
- **摘要**：新披露的 CVE-2026-31431（代号 Copy Fail）是 Linux 内核 authencesn 加密模板中的漏洞，影响近 17 年的所有 Linux 内核版本。一个 732 字节的 Python 脚本即可在 Ubuntu、Amazon Linux、RHEL、SUSE 上实现本地提权到 root。
- **影响**：如果你的 Agent 服务部署在 Linux 服务器上，需要立即排查和打补丁。这个漏洞的利用门槛极低。
- **行动建议**：立即检查服务器内核版本，应用最新安全补丁。
