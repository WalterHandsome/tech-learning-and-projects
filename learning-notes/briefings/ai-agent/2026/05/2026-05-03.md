### 1. Open Design：开源版 Claude Design，用编码 Agent 做设计引擎

- **来源**：[nexu-io/open-design](https://github.com/nexu-io/open-design)
- **标签**：`开源` `Claude Design` `AI 设计` `编码 Agent`
- **评分**：时效 5 / 一手 5 / 相关 5 / 实用 5 = 总分 20
- **摘要**：Anthropic 4 月 17 日发布 Claude Design 后，nexu-io 迅速推出开源替代方案 Open Design。支持 12 种编码 Agent CLI（Claude Code、Codex、Cursor、Gemini CLI、Kiro 等），提供 31 个可组合 Skill 和 72 套品牌级设计系统。本地优先、BYOK、可部署到 Vercel，支持 HTML/PDF/PPTX/MP4 导出。HN 获得 192 分和 87 条评论。
- **影响**：将 AI 编码 Agent 的能力从"写代码"扩展到"做设计"，开发者可以用熟悉的 Agent 工具链直接生成原型、幻灯片和设计稿。这是 Agent 能力边界的重要扩展。
- **行动建议**：如果团队有快速原型需求，值得试用；关注其 Skill 系统的设计模式，可借鉴到自己的 Agent 工作流中

### 2. Agent Harness 应该放在沙箱外面——多用户 Agent 架构的关键决策

- **来源**：[The agent harness belongs outside the sandbox](https://www.mendral.com/blog/agent-harness-belongs-outside-sandbox)
- **标签**：`Agent 架构` `安全` `多用户` `生产部署`
- **评分**：时效 5 / 一手 5 / 相关 5 / 实用 4 = 总分 19
- **摘要**：Mendral 团队分享了构建多用户 Agent 系统的架构决策：将 Agent 驱动循环（harness）放在沙箱外部运行。核心优势包括：凭证不进入沙箱、沙箱可按需挂起/恢复、沙箱变成"牲畜"而非"宠物"。文章详细讨论了持久化执行（基于 Inngest）、分布式文件系统问题和多用户共享记忆的解决方案。HN 获得 76 分和 61 条评论。
- **影响**：这是目前最清晰的多用户 Agent 生产架构分析之一。对于正在构建 Agent 平台的团队，harness 内外的选择直接影响安全模型、成本和可扩展性。
- **行动建议**：如果在构建多用户 Agent 系统，这篇是必读参考；单用户场景可以先用 harness-inside 模式，但提前考虑迁移路径

### 3. AI 招聘系统的自我偏好：实证研究揭示 AI 评估 AI 生成简历时的偏见

- **来源**：[AI Self-preferencing in Algorithmic Hiring](https://arxiv.org/abs/2509.00462)
- **标签**：`AI 偏见` `招聘` `论文` `伦理`
- **评分**：时效 5 / 一手 5 / 相关 4 / 实用 3 = 总分 17
- **摘要**：研究发现 AI 招聘系统在评估简历时存在"自我偏好"现象——倾向于给 AI 生成的简历更高评分。HN 上获得 319 分和 170 条评论，是今日讨论最热烈的 AI 话题之一。随着 AI Agent 越来越多地参与决策流程，这类偏见问题值得所有 Agent 开发者关注。
- **影响**：当 AI Agent 被用于评估其他 AI 的输出时，可能产生系统性偏见。这对 Agent 在 HR、内容审核、代码审查等场景的应用提出了警示。
- **行动建议**：在设计 Agent 评估流程时，加入人类审核环节和偏见检测机制

## 📄 值得关注的论文

### ARTIST: 通过强化学习实现 LLM 的 Agent 推理与工具集成

- **作者/机构**：Joykirat Singh 等（Microsoft Research 参与）
- **摘要**：提出 ARTIST 框架，将 Agent 推理、强化学习和工具集成紧密耦合。模型可自主决定何时、如何、调用哪个工具，通过基于结果的 RL 学习工具使用策略，无需步骤级监督。
- **关键贡献**：在数学推理和多轮函数调用基准上，比基线模型提升最高 22%。证明 Agent RL + 工具集成是 LLM 问题解决的新前沿。
- **链接**：[arXiv:2505.01441](https://arxiv.org/abs/2505.01441)

### AI 招聘中的自我偏好：实证证据与洞察

- **作者/机构**：详见论文
- **摘要**：实证研究发现 AI 系统在评估简历时倾向于偏好 AI 生成的内容，揭示了算法招聘中的系统性偏见。
- **关键贡献**：首次大规模实证验证 AI 自我偏好现象，对 Agent 在决策场景的应用有重要警示意义。
