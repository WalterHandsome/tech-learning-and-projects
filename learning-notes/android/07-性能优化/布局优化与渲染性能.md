## 1. 布局层级优化

```xml
<!-- ❌ 嵌套过深 -->
<LinearLayout>
    <LinearLayout>
        <RelativeLayout>
            <TextView />
        </RelativeLayout>
    </LinearLayout>
</LinearLayout>

<!-- ✅ 使用 ConstraintLayout 扁平化 -->
<ConstraintLayout>
    <TextView
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintTop_toTopOf="parent" />
</ConstraintLayout>

<!-- merge 减少层级 -->
<merge xmlns:android="http://schemas.android.com/apk/res/android">
    <TextView android:id="@+id/title" ... />
    <ImageView android:id="@+id/icon" ... />
</merge>

<!-- ViewStub 延迟加载 -->
<ViewStub
    android:id="@+id/errorStub"
    android:layout="@layout/layout_error"
    android:layout_width="match_parent"
    android:layout_height="wrap_content" />
```

```kotlin
// 按需加载 ViewStub

// 或

```

## 2. 过度绘制检测

```kotlin
// 开发者选项 → 调试 GPU 过度绘制
// 蓝色: 1x 过度绘制（可接受）
// 绿色: 2x（注意）
// 粉色: 3x（需优化）
// 红色: 4x+（必须优化）

// 优化方法：
// 1. 移除不必要的背景
// 2. 使用 clipRect 限制绘制区域
class OptimizedView(context: Context) : View(context) {
    override fun onDraw(canvas: Canvas) {
        canvas.save()
        canvas.clipRect(visibleRect)  // 只绘制可见区域
        super.onDraw(canvas)
        canvas.restore()
    }
}
```

## 3. RecyclerView 优化

```kotlin
recyclerView.apply {
    // 固定大小
    setHasFixedSize(true)

    // 预取
    (layoutManager as LinearLayoutManager).initialPrefetchItemCount = 4

    // 缓存
    setItemViewCacheSize(20)

    // 共享 RecycledViewPool
    recycledViewPool = sharedPool

    // 避免嵌套滚动冲突
    isNestedScrollingEnabled = false
}
```

## 4. Compose 重组优化

```kotlin
// 1. 使用 Stable/Immutable 标注
@Immutable
data class UserUiModel(
    val id: String,
    val name: String,
    val avatar: String
)

// 2. 避免不必要的重组
@Composable
fun UserList(users: List<UserUiModel>) {
    LazyColumn {
        items(users, key = { it.id }) { user ->
            // key 帮助 Compose 识别项目，减少重组
            UserItem(user)
        }
    }
}

// 3. remember 缓存计算
@Composable
fun ExpensiveScreen(items: List<Item>) {
    val sorted = remember(items) { items.sortedBy { it.name } }
    val grouped = remember(sorted) { sorted.groupBy { it.category } }
}

// 4. derivedStateOf 减少重组
@Composable
fun ScrollableList() {
    val listState = rememberLazyListState()
    val showButton by remember {
        derivedStateOf { listState.firstVisibleItemIndex > 5 }
    }
    // showButton 只在跨越阈值时触发重组
}

// 5. Lambda 稳定性
@Composable
fun ParentScreen(viewModel: MyViewModel = hiltViewModel()) {
    // ❌ 每次重组创建新 lambda
    ItemList(onClick = { id -> viewModel.onItemClick(id) })

    // ✅ 使用方法引用
    ItemList(onClick = viewModel::onItemClick)
}
```
